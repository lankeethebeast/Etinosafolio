# Onaifo Etinosa Favour - Portfolio

A premium portfolio website for Onaifo Etinosa Favour (Lxnkee), a Full Stack Developer specializing in building modern web experiences.

## 🚀 Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript
- **Fonts:** [Outfit](https://fonts.google.com/specimen/Outfit) via Google Fonts
- **Icons:** Custom SVGs
- **Email:** [EmailJS](https://www.emailjs.com/)

## 📂 Project Structure
```text
.
├── assets/
│   └── images/      # Project images and profile photos
├── css/
│   └── styles.css   # Main stylesheet
├── js/
│   └── script.js    # Interactivity and logic
├── index.html       # Entry point
├── .gitignore       # Git ignore rules
└── README.md        # Project documentation
```

## 🛠️ Setup
1. Clone the repository
2. Open `index.html` in any modern web browser
